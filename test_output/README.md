# Папка test_output/

## Назначение
Логи, результаты тестирования и промежуточные выгрузки.

## Основные файлы
- `main.log` - основной лог
- `database.log` - лог работы с БД  
- `errors.log` - лог ошибок

## Принципы
- Логи перезаписываются при каждом запуске
- Папка может очищаться для экономии места
