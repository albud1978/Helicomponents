# Архив проекта прогнозирования оборота ВНВ (CPU версия)

## Описание
Это архивная версия проекта для прогнозирования оборота воздушных винтов (ВНВ) на вертолетах с использованием CPU вычислений.

## Структура проекта

### Основные файлы
- `cycle_full_CPU.py` - Основной модуль обработки циклов на CPU
- `cycle_full_GPU.py` - GPU версия обработки циклов
- `analytic_CPU.py` - Аналитические функции
- `load_helicopters.py` - Загрузка данных о вертолетах
- `run_cycle.py` - Запуск циклов обработки
- `test_data_consistency.py` - Тестирование консистентности данных

### Jupyter Notebooks
- `OLAP-initial.ipynb` - Начальная OLAP обработка
- `Virtual.ipynb` - Виртуальная обработка данных
- `Status_P Initial.ipynb` - Обработка статусов
- `Counters_day1.ipynb` - Счетчики за день
- `Program.ipynb` - Программная логика
- `Agregat.ipynb` - Агрегатная обработка
- Другие аналитические ноутбуки

### Папки данных
- `Helicopters/` - Данные о вертолетах
- `Выгрузка/` - Выгрузки данных ВНВ
- `Выгрузка ВС/` - Выгрузки данных воздушных судов
- `Аналитика/` - Аналитические данные (включая MultiBOM)
- `Агрегат/` - Агрегатные данные
- `Программа/` - Программные данные

## Технологии
- Python с использованием pandas, numpy
- ClickHouse для хранения данных
- Jupyter Notebooks для анализа
- CPU-based вычисления
