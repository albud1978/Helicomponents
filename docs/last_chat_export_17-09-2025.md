# Экспорт чата от 17-09-2025

## Основные темы чата
- Интеграция спавна Mode B в Mode A, участие новорождённых в квотах 3→2
- FRAMES‑upfront, индексация MP3∪MP5, проблема intent_flag=0 у новорождённых

## Решённые задачи
- Уточнена архитектура FRAMES: frames_union_no_future = |MP3_planes ∪ MP5_planes| (279+7=286)
- Выравнивание спавна по индексам MP5: frames_initial = frames_union_no_future, next_idx = first_future_idx

## Проблемы и их решения
- Новые агенты имели idx вне диапазона FRAMES → не ставился intent (несоответствие 293 vs idx≈7113). Исправлено: сдвиг индексов на хвост FRAMES (без дырок).
- Остаточная проблема: intent_flag у новорождённых остаётся 0 после D+1. Гипотеза: рассогласование FRAMES при сборке RTC или порядок слоёв/момент экспорта. Отмечено как приоритет.
